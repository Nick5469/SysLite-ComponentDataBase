## 开始使用

> [!NOTE]
>
> **详细使用说明请参阅 [Component DataBase 文档]()**
>
> 对于普通用户，推荐使用[SysLite]()系统定制工具进行组件操作

**确保您的设备满足以下要求**

- 目标操作系统 Windows Vista 及以上[^1]

[^1]: 数据库API在调用时要求指定目标操作系统路径（即欲读取组件信息的系统路径）

> [!IMPORTANT]
>
> 在读取数据库信息前，**请先对照本地数据库的HASH值与发布页面上的是否一致**，避免数据库信息出错所导致的问题
